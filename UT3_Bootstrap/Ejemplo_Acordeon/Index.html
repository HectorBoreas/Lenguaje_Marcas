Acordeón descripción

Acordeón: componente de boostrap que nos permite crear un menú desplegable, en el cual a primera vista tenemos unos botones que al pinchar en ellos se despliega la información correspondiente a cada botón.

Atributos CSS

Como parte del enfoque de variables CSS en evolución de Bootstrap.s, los acordeones ahora utilizan variables CSS locales en .accordion para una mayor personalización en tiempo real. Los valores de las variables CSS se establecen a través de Sass, por lo que la personalización de Sass también se admite.
